Tables

client.config
The clients configuration (Most from Discordia client)

Fields:

  • prefix string The prefix the bot will use default '!'
  • defaultHelp boolean Whether to use the default help or not default true
  • owners table The owners of this bot default {}
  • routeDelay number Minimum time in milliseconds to wait between HTTP requests per-route default 300
  • maxRetries number The maximum number of retries to attempt after an HTTP request fails default 5
  • shardCount number The total number of shards the application is using (0 signals to use the recommended count) default 0
  • firstShard number The ID of the first shard to run on the client (0 is the minimum) default 0
  • lastShard number The ID of the last shard to run on the client (-1 signals to use shardCount - 1) default -1
  • largeThreshold number Limit to how many members are initially fetched per-guild on start-up default 100
  • cacheAllMembers boolean Whether to cache all members (If false, offline members may not be cached) default false
  • autoReconnect boolean Whether to attempt to reconnect after an unexpected gateway disconnection default true
  • compress boolean Whether to allow for compressed gateway payloads default true
  • bitrate number The default bitrate to use for voice connections, from 8000 to 128000 default 64000
  • logFile string The file to use for logging default 'discordia.log'
  • logLevel string The level to use for logging (see Enumerations) default logLevel.info
  • dateTime string The date and time format to use logging default '%F %T'
  • syncGuilds boolean Whether to automatically sync all guilds on start up (user-accounts only) default false
