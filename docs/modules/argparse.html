Methods
new (tbl, command)
Construct an argument parser

Parameters:

  • tbl argument[] The argument table to parse
  • command command The command which you are using this on

Tables

argument
An argument

Fields:

  • id string The key of where to put the argument
  • type string The type of the argument
  • default string or function The default argument, if passed a function it will be passed the argument and message default nil

Fields

int
The built in types; They are automatically type casted to what they are named
  • int Check if the input is a number
  • string Does nothing as all inputs are strings
  • boolean Check if the input is true or false
  • ban Check if the input is a valid ban
  • channel Check if the input is a channel; does not allow searching
  • emoji Check if the input is a emoji
  • guild Check if the input is a guild, will preform a search if the input is a string
  • invite Check if the input is a invite code
  • member Check if the input is a member; does not allow searching
  • message Check if the input is message jump link or an id
  • role Check if the input is a role, will preform a search if the input is a string
  • textchannel Check if the input is a text channel, will preform a search if the input is a string
