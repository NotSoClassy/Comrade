Class Template

A template for creating embeds and more templates

Methods

Template\new (start, useEtLua) Create a new template
Template\render (env) Render the template into an embed
Template\construct (env) Render the template into another template

Methods

Template\new (start, useEtLua)
Create a new template

Parameters:

  • start table The starting point of the embed
  • useEtLua boolean To use etlua as the renderer, mustache is the default renderer
Template\render (env)
Render the template into an embed

Parameters:

  • env table The table to pass to the renderer
